Redis

  • 非关系型数据库
  • Redis（Remote Dictionary Server )，即远程字典服务 !
    是一个开源的使用ANSI C语言编写、支持网络、可基于内存亦可持久化的日志型、Key-Value数据库，并提供多种语言的API。

五大数据类型

String

  • 基本指令：
    • set + “name”+“value” 插入
    • get +“name” 获取对应值
    • EXISTS +“name”判存在
    • APPEND + “name” +“value” 将value拼接到name的值的后面
    • STRLEN + “name” 获取长度
    • incr + “name”自增
    • decr + “name”自减
    • GETRANGE + ”name“ int int，截取
    • GETRANGE +”name“ int int，截段替换
